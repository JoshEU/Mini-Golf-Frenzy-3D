# Mini Golf Frenzy 3D #

## About: ##  
Mini Golf Frenzy 3D is a casual mini-golf game built for mobile which is set in a fantasty, space themed world. There are 3 fun and exciting levels to be played, ranging from Easy to Hard in difficulty.  
Can you make it to the final level and become the Mini Golf Frenzy Champion? Come and Find Out! :D
